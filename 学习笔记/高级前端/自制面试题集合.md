#### 1 js的数据类型、转换规则、类型检测

```
基本类型有:
   number
   string
   boolean
   null
   undefined
   symbol： 唯一值，作用 保证标志的唯一性。底层用到了(hasInstance iterator....等)
   bigInt: 大数
   
   对象类型：
      标准普通对象 Object
      标准特殊对象 Array/regexp/Date/Error/Math....
      非标准特殊对象 Number/String/Boolean.....基于构造函数[或者Object]创造出来的原始值对象类型的格式信息，类型属于对象类型
```
